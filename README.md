A Campaign made of Classes

College of Staten Island
CSC330 Object Orientation Fall 2024 with Prof. Tatiana Anderson
Project By: Bailey Perillo, Amberlee Aponte, Ilir Krasniqi

We created a DnD tracker, used for any DnD players that want a virtual way to keep track of their current campaigns so they won't forget what happened during their next meetup!
